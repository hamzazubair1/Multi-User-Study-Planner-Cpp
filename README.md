# Simple Study Planner (C++)

A console-based study planner application developed in C++.  
This program helps students manage their study tasks efficiently by using basic programming concepts and file handling.

## Features
- Add new study tasks
- View all tasks
- View pending tasks only
- Mark tasks as completed
- Suggest next task based on priority and due date
- Save tasks to a file
- Load tasks automatically on program start

## Concepts Used
- Arrays
- Functions
- Conditional statements
- Loops
- File handling (`fstream`)
- Input validation

## How to Run
1. Open the project in any C++ compiler (Dev C++, Code::Blocks, Visual Studio, etc.)
2. Compile and run the program
3. Follow the menu options shown in the console

## File Storage
- Tasks are saved in a file named `tasks.txt`
- Data is automatically loaded when the program starts
